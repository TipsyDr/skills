# Source Notes

This repository is a local curated snapshot of skills installed or developed in a Codex workspace.

## Included

- Skill instructions in `SKILL.md`
- Small helper scripts used by the skills
- Public/template-safe reference notes used by the skills
- `agents/openai.yaml` display metadata

## Excluded

- Codex plugin cache folders
- Obsidian vault content
- Conversation transcripts
- Generated manuscripts
- Local machine cache/state
- API keys, tokens, secrets, and private config files

## Recency

Some webnovel references describe platform voice, writing patterns, or 2026-era meme snapshots. Treat them as writing guidance, not live market data. Verify live rankings, trends, and policy changes before making time-sensitive decisions.
