# 男频写作经典学习库

Source check date: 2026-05-19

## 使用边界

- 本库用于男频小说写作学习：结构、节奏、人物关系、场景功能、语言气势、叙事策略。
- 现代网文与现代文学只保存公开资料层面的分析、官方来源链接、阶段化学习卡和可迁移写法；不保存章节正文、长段摘录、盗版链接或可替代原作阅读的复述。
- 古籍与公版文本也不做原文堆砌，统一转成“男频写作能力”调用卡。
- 如果用户提供合法章节笔记，可把现代网文从 `source-limited-complete` 升级为 `deep-complete`。

## 调用示例

- 要写“寒门/赘婿/小人物起步，但后期要推到国家格局”：先读《赘婿》作品卡，再读 `small-cut-to-big-scope`。
- 要写“朝堂辩论、君臣互相试探、国势压人”：读《绍宋》作品卡、`historical-pressure-family-state` 和 `political-game`。
- 要写“每章有案子、笑点、升级爽点”：读《大奉打更人》作品卡和 `investigation-comedy-rhythm`。
- 要写“人物一出场就像名场面”：读《雪中悍刀行》作品卡和 `jianghu-ensemble-famous-scenes`。
