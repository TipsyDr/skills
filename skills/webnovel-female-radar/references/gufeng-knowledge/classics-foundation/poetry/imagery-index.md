# 诗词意象与情绪调用库

Last updated: 2026-05-19

## 使用方式

本库按写作场景调用，不逐首保存原文。需要标题、氛围、人物情绪或古风句式时，先选主题，再选意象，再决定句式密度。

## 标题生成原则

- 宅斗标题：物件 + 权力动作，如“旧钥”“夜账”“听帘”“分香”。
- 复仇标题：冷意象 + 因果动作，如“雪债”“灯下证”“归刃”。
- 探案标题：物证 + 场所，如“簪痕”“药灯”“案上灰”。
- 爱情标题：季节 + 未竟动作，如“春迟”“月不归”“梦里书”。

## 古风句式原则

| 场景 | 句式 |
|---|---|
| 含蓄情感 | 先物后人，先动作后心意。 |
| 权谋辩论 | 先利害，再旧例，最后后果。 |
| 世家日常 | 先称呼、座次、流程，再暗流。 |
| 奇情幻梦 | 先日常，后一处异常，最后让异常回到人心。 |
| 悲壮转折 | 先给人物选择，再给不可挽回的代价。 |

## 误用警告

- 不要用诗词替代剧情。
- 不要全篇堆“月、花、香、泪”。
- 不要把古典女性声音写成单纯柔弱；很多诗词里的克制本身就是力量。
