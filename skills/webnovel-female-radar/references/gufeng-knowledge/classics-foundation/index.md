# 古籍根基库

Last updated: 2026-05-19

## 使用边界

本库面向女频古风写作训练，优先把公开古籍转化为可调用的场景方法、人物关系、语言节奏和结构模型。不保存整章、整篇、整首原文，不保存盗版资源，不把学习卡写成可替代原典阅读的内容。

主来源：Chinese Text Project 中国哲学书电子化计划公开页面与 API。中华经典古籍库作为有权限后的校勘来源，不假定当前可访问。

## 分层精读口径

| 类型 | 粒度 | 输出形式 |
|---|---|---|
| 《红楼梦》 | 120回逐回学习卡 | 每回一行：位置、功能、关系、权力、情绪/语言、女频转化、禁用误区 |
| 史传/诸子叙事 | 按人物传、事件、外交/谋略篇 | 结构模型和话术模型 |
| 戏曲/传奇 | 按折、出、单元故事 | 爱情阻力、梦境、奇情反转和角色弧线 |
| 诗词 | 按意象与情绪主题群 | 场景、人物情绪、标题与古风句式调用 |
| 《金瓶梅》 | 世情/家庭权力/日常细节 | 只学生活质感和人情结构，避开低俗桥段 |

## 核心原则

- 学《红楼梦》：学群像、权力暗流、生活细节、含蓄情感，不复制人物和桥段。
- 学《世说新语》：学短笔人物、对话锋芒、士族姿态，不把人物写成空泛装冷。
- 学《史记》《左传》《战国策》：学命运转折、盟约政治、话术博弈，不把权谋写成口号。
- 学诗词戏曲：学意象和情绪推进，不用诗词替代剧情。
- 学《金瓶梅》：学世情网络、账目、馈赠、宴饮、家内权力，不学习低俗桥段。
