# 古诗词与古风语言库

Last updated: 2026-05-19

## 使用原则

古风语言不是堆砌诗词，而是用意象、节奏和礼法动作承载情绪。写女频古言时，语言优先服务人物处境：闺阁写边界，庭院写秩序，雨雪写阻隔，灯窗写等待，药香写身体和隐痛，案牍写职业压力。

## 意象组

| 意象 | 适用场景 | 情绪功能 | 写法提醒 |
|---|---|---|---|
| 闺阁 | 少女成长、嫡庶压迫、婚事议论 | 有边界的安全和窒息 | 少写“美”，多写窗、帘、门、座次和传话。 |
| 雨雪 | 离别、失势、归来、赶路 | 阻隔、洗净、冷意、命运感 | 雨雪不要只当背景，要影响行动和信息传递。 |
| 灯窗 | 等待、夜谈、病中、查案 | 私密、疲惫、守望 | 灯下最适合写未出口的话和未查明的事。 |
| 药香 | 医女、病弱、伤后、复仇冷感 | 身体记忆、隐痛、专业感 | 药名不宜乱堆，优先写气味、器具和流程。 |
| 簪钗 | 身份、婚嫁、女性评价 | 名分、体面、束缚 | 首饰是社会符号，不只是装饰。 |
| 庭院 | 家宅权力、请安、责罚 | 秩序和等级 | 用站位、路程、门槛、廊下等待写权力。 |
| 离别 | 远嫁、出府、逃亡、赴任 | 选择成本 | 离别场景要带走一样资源或关系。 |
| 婚嫁 | 交易、救赎、权力联盟 | 身份重置 | 婚礼不是甜点本身，是女主进入新规则。 |
| 仕途 | 男主/家族权力、朝局压力 | 外部秩序压入内宅 | 朝局只写和女主选择有关的部分。 |
| 案牍 | 探案、医案、账本、契约 | 证据、职业、理性 | 案牍场景要让读者看见女主的方法。 |

## 句式原则

- 宅斗：短动作 + 礼法名词 + 旁人反应。例：先写谁坐着、谁站着、谁递话，再写情绪。
- 重生：少解释前世，多写女主对旧细节的提前警觉。
- 复仇：句子宜锋利，情绪宜收住；让证据替女主发声。
- 探案：线索句短，氛围句长；关键推理不要被辞藻遮住。
- 职业女主：专业动作必须准确稳定，不能只用“她很厉害”概括。

## 古诗词转化原则

| 诗词资源 | 可学内容 | 转化方法 |
|---|---|---|
| 李清照词 | 闺阁、离愁、物候中的女性自我 | 学“细物承情”，不要照搬句子；用帘、花、酒、雨写心境变化。 |
| 温庭筠/花间词 | 色彩、妆饰、晨昏氛围 | 学镜头感和物象密度，避免把女主写成静态装饰。 |
| 晏几道/秦观 | 旧情、梦境、错过 | 适合写误会后回望，但要让人物选择推动剧情。 |
| 杜甫/王维 | 家国、旅途、空间开阔 | 用于把内宅情绪推向外部世界。 |
| 唐传奇/笔记小说 | 奇案、异闻、因果感 | 适合探案或复仇氛围，但案件逻辑仍要清楚。 |

## 场景调用模板

| 场景 | 语言策略 |
|---|---|
| 女主被长辈敲打 | 先写茶盏、座次、称呼，再写话中刺。 |
| 女主夜查线索 | 写灯火范围、纸页声、门外动静、手指停顿。 |
| 医女诊病 | 写望闻问切的顺序、病人遮掩、药气变化。 |
| 婚事被议 | 写屏风内外、媒人话术、亲族眼色。 |
| 重生后重见旧人 | 写女主身体反应和克制动作，不急着揭前世。 |
| 复仇前夜 | 写清点证据、销毁退路、整理衣饰，突出不可逆。 |

## 禁忌

- 不为了“古”而乱用生僻词。
- 不用诗词替代剧情推进。
- 不把女性角色全写成花、月、香、泪；意象要跟身份和行动绑定。
- 不复制现代影视台词式古风。
