# 公共平台语感资料

Last updated: 2026-05-18

来源：复制自男频 skill 的公共平台语感资料，可与 `webnovel-hot-memes` 同步更新。这里只保存平台表达方法，不绑定男频或女频作品。

## 可复用表面

- 弹幕：适合即时震惊、验操作、战力/关系判断、身份怀疑。
- 评论区：适合站队、争议、复盘、玩梗和读者二创。
- 短视频复盘：适合三段式标题、强钩子、反转解释和爽点剪辑。
- 群聊/论坛：适合机制讨论、角色关系推理、榜单和更新节奏吐槽。

## 女频常用转化

- CP 站队：谁更值得、谁在隐忍、谁先动心、谁在自我感动。
- 女主人设维护：事业脑、清醒、嘴硬心软、被误解但不自证。
- 情绪复盘：名场面、追妻节点、身份揭露、家庭压迫、权谋反杀。
- 评论区争议：男主是否合格、配角是否越界、家人是否吸血、女主是否该原谅。

## 使用边界

- 不把短期热点硬塞进严肃正文。
- 不让所有角色共享同一种网络语气。
- 涉及现实人物、灾难、地域和敏感公共事件时默认不入库。
